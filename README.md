## What It Does

**shorthand-mem** compresses verbose documents into a structured symbolic notation (SHM) and reconstructs them on demand. It runs entirely inside Claude ‚ÄĒ install the `.skill` file, type commands in chat, everything else is handled.

No Python. No API key. No local server. Just Claude.

```
You type:  SQUEEZE PRD <your document>
Claude:    ‚Üí compresses to SHM notation
           ‚Üí runs fidelity check (dates, amounts, entities captured)
           ‚Üí saves to memory + exports .shm file
           ‚Üí 78% fewer tokens next time you need this context
```

## Token Reference

### Universal ‚ÄĒ All Domains

| Token | Meaning |
|---|---|
| `OBJ(Name)` | Core entity, system, module, or actor |
| `SCOPE: Name` | Module or domain boundary |
| `GOAL(Action)` | Primary intent |
| `REQ: action()` | Strict functional mandate |
| `RULE: constraint` | Business logic, rate limit, or guardrail |

## Full Example: PRD

### Input (~880 tokens)

> The Notification Delivery Service routes alerts across in-app, email, and push. Critical notifications must be delivered within 5 seconds; Standard within 60 seconds. Email via SendGrid ‚ÄĒ on 5xx or 3-second timeout, retry 3√ó with exponential backoff from 500ms. After 3 failures ‚Üí DeadLetterQueue + PagerDuty. Push via FCM ‚ÄĒ if `push_enabled=false`, skip silently. FCM token-expired ‚Üí delete stale token + skip. Hard dependency on UserPreferences ‚ÄĒ if unavailable, suspend all and queue for replay. Rate limit: 50 notifications/user/hr. Excess silently dropped + EventLog entry. Critical failures escalate to engineering lead.

### Output (~195 tokens ‚ÄĒ 78% reduction)

```shm

OBJ(Notification_Delivery_Svc)
GOAL: Multi_channel_alert_routing(in-app, email, push)
IF (priority == Critical) ‚Üí THEN (deliver <= 5s)
IF (priority == Standard) ‚Üí THEN (deliver <= 60s)
REQ: Email_via_SendGrid
ERR(SendGrid_5xx || timeout>3s) ‚Üí RETRY x3 | backoff(500ms, exponential)
IF (retry_count == 3 && fail) ‚Üí WRITE(DeadLetterQueue) + PAGE(PagerDuty)
RULE: user.push_enabled == false ‚Üí SKIP silently
ERR(FCM_token_expired) ‚Üí DELETE(UserDevices.token) + SKIP
DEP(UserPreferences_Svc)++ ‚Üí IF (unavailable) ‚Üí SUSPEND_ALL + QUEUE_replay
REQ: RateLimit == 50 notifications/user/hr
IF (rate_exceeded) ‚Üí DROP silently + WRITE(EventLog.RateLimitExceeded)
RISK(Critical_delivery_failure)++ ‚Üí ESCALATE(engineering_lead)
```
